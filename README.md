# Ping Ball
A 2D game where you have to keep at least one ball bouncing off the paddle to keep the game going. You get 1 point for each bounce, and every 5 points a new ball appears in the middle of the screen.

## Round of Changes

### First Round
The logic of the game is implemented with basic shapes, scripts, prefabs, collisions, and 2D physics materials.

### Second Round
A big update in terms of visuals with the addition of a menu and simple animations.
- **Note:** In this second round of changes, the most recent version of the game is on the `GameWithMenu` branch because of conflicts on trying to merge the `test` branch (has a prototype of the menu) with `main` (has the logic of the game). To avoid any more conflicts, a third `GameWithMenu` branch was created for this purpose.
